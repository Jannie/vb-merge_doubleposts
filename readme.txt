Description:
------------

vBulletin addon to merge 2 sequent posts of the same author.

Installation:
-------------

1. Upload files

2. Import product-mergedoubleposts.xml. That's enougth for english version.

3. If you plan to use splitter, and need russian language there,
   import ALSO product-mergedoubleposts_ru.xml. You need BOTH xmls imported.
   Second one contains ONLY translation deals.

Not yet released at vborg.
